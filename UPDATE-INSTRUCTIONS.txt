UPLOAD TO GITHUB
1. Unzip this package.
2. Open the extracted folder.
3. In your GitHub repository, select Add file > Upload files.
4. Upload EVERY file inside this folder, including all JPG and PNG files.
5. Allow index.html, styles.css and app.js to replace the existing files.
6. Commit the changes.
7. Wait 2–5 minutes, then open the live website and press Ctrl + F5.
